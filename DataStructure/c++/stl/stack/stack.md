# Stack

## Declare

```c++
stack<int> s1; 
stack<string> s2;
```

## Push/Insert value

```c++
stack<int> stack;
stack.push(21);
```

## Pop/Remove value

```c++
// it does not return value
stack.pop();
```

## Get top value

```c++
int val = stack.top();
```

## Check stack empty

```c++
if(stack.empty()){
}
```

## Check stack size

```c++
stack.size()
```
